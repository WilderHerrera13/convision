# Plan 11-06: Frontend — New Consultation Flow (4 Tabs)

**Phase:** 11 — Specialist Appointment & Clinical History Flow
**Plan:** 06 of 08
**Wave:** 6
**Depends on:** 11-05 (appointment detail page must exist; clinical record API endpoints must be live)

---

## Goal

Build the 4-tab new consultation form at `/specialist/appointments/:id/clinical-history`
(Figma nodes 2226:1416, 2227:1472, 2228:1528, 2228:1678). Each tab auto-saves
to the backend on navigation. The shared layout includes the `ClinicalAsidePanel`
(patient card + vigente Rx + DX activo + step counter + legal tip).

---

## Success Criteria

- [ ] Route `/specialist/appointments/:id/clinical-history` renders `ClinicalHistoryNewConsultationPage`
- [ ] Page layout: sidebar 240px + topbar 60px + FormCard 780px + AsidePanel 332px (matches Figma layout)
- [ ] Tab 1 (Anamnesis): motivo consulta textarea + 4 antecedentes sections + corrección actual selects — matches Figma 2226:1416
- [ ] Tab 2 (Examen Visual): AV table (sc/cc × dist/near × OD/OI) + refracción obj/subj + keratometría + PIO + biomicroscopía + motilidad — matches Figma 2227:1472
- [ ] Tab 3 (Diagnóstico): CIE-10 autocomplete + tipo radio + diagnósticos relacionados + RIPS preview card + plan de atención — matches Figma 2228:1528
- [ ] Tab 4 (Prescripción): formula table OD/OI + tipo lente/material/uso + vigencia + firma block — matches Figma 2228:1678
- [ ] Navigating between tabs triggers auto-save of current tab data (PUT per step endpoint)
- [ ] Tab completion dots appear when step data exists
- [ ] "Firmar y completar consulta" button on Tab 4 navigates to `PrescriptionPreviewModal` (Plan 11-08)
- [ ] `npm run build` passes with 0 TypeScript errors

    ```ts
    import api from './api';

    export const createClinicalRecord = (appointmentId: number, recordType: 'new_consultation' | 'follow_up') =>
      api.post(`/appointments/${appointmentId}/clinical-record`, { record_type: recordType });

    export const getClinicalRecord = (appointmentId: number) =>
      api.get(`/appointments/${appointmentId}/clinical-record`);

    export const upsertAnamnesis = (appointmentId: number, data: AnamnesisInput) =>
      api.put(`/appointments/${appointmentId}/clinical-record/anamnesis`, data);

    export const upsertVisualExam = (appointmentId: number, data: VisualExamInput) =>
      api.put(`/appointments/${appointmentId}/clinical-record/visual-exam`, data);

    export const upsertDiagnosis = (appointmentId: number, data: DiagnosisInput) =>
      api.put(`/appointments/${appointmentId}/clinical-record/diagnosis`, data);

    export const upsertPrescription = (appointmentId: number, data: PrescriptionInput) =>
      api.put(`/appointments/${appointmentId}/clinical-record/prescription`, data);

    export const signClinicalRecord = (appointmentId: number, professionalTp: string) =>
      api.post(`/appointments/${appointmentId}/clinical-record/sign`, { professional_tp: professionalTp });
    ```
