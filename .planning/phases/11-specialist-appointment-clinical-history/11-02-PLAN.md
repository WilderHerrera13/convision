    2. **Update `Take`** to:
       - Check if specialist already has an `in_progress` appointment via `GetActiveBySpecialist`
       - If found, return a new `ErrActiveAppointment{ActiveID: active.ID}` (HTTP layer maps to 409)
       - Set `StartedAt = &now` when transitioning to `in_progress`

    Updated `Take`:
    ```go
    func (s *Service) Take(id uint, specialistID uint) (*domain.Appointment, error) {
        // Conflict guard
        existing, err := s.repo.GetActiveBySpecialist(specialistID)
        if err == nil && existing != nil && existing.ID != id {
            return nil, &ErrActiveAppointment{ActiveID: existing.ID}
        }

        a, err := s.repo.GetByID(id)
        if err != nil {
            return nil, err
        }
        if a.Status != domain.AppointmentStatusScheduled && a.Status != domain.AppointmentStatusPaused {
            return nil, &domain.ErrValidation{Field: "status", Message: "appointment cannot be taken in current state"}
        }

        now := time.Now()
        a.Status = domain.AppointmentStatusInProgress
        a.TakenByID = &specialistID
        if a.StartedAt == nil {
            a.StartedAt = &now
        }

        if err := s.repo.Update(a); err != nil {
            return nil, err
        }
        return s.repo.GetByID(a.ID)
    }
    ```

    3. **Add `Complete`**:
    ```go
    func (s *Service) Complete(id uint) (*domain.Appointment, error) {
        a, err := s.repo.GetByID(id)
        if err != nil {
            return nil, err
        }
        if a.Status != domain.AppointmentStatusInProgress && a.Status != domain.AppointmentStatusPaused {
            return nil, &domain.ErrValidation{Field: "status", Message: "appointment is not in progress or paused"}
        }

        now := time.Now()
        a.Status = domain.AppointmentStatusCompleted
        a.CompletedAt = &now

        if err := s.repo.Update(a); err != nil {
            return nil, err
        }
        return s.repo.GetByID(a.ID)
    }
    ```

    4. **Add `GetActive`**:
    ```go
    func (s *Service) GetActive(specialistID uint) (*domain.Appointment, error) {
        return s.repo.GetActiveBySpecialist(specialistID)
    }
    ```

    5. **Add `ErrActiveAppointment` struct** in the same file:

    ```go
        ActiveID uint
    }
    func (e *ErrActiveAppointment) Error() string {
        return fmt.Sprintf("specialist already has active appointment %d", e.ActiveID)
    }
    ```


    `GetByAppointmentID` must preload `Anamnesis`, `VisualExam`, `Diagnosis`, `Prescription`, `FollowUp`.

    **AnamnesisRepository**:
    ```go
    func (r *AnamnesisRepository) GetByClinicalRecordID(recordID uint) (*domain.Anamnesis, error)
    func (r *AnamnesisRepository) Upsert(a *domain.Anamnesis) error
    ```
    `Upsert`: use `db.Where("clinical_record_id = ?", a.ClinicalRecordID).Assign(*a).FirstOrCreate(a)` pattern,
    or `db.Save(a)` only if ID is already set (first check by clinical_record_id).
    Preferred safe pattern:
    ```go
    var existing domain.Anamnesis
    err := r.db.Where("clinical_record_id = ?", a.ClinicalRecordID).First(&existing).Error
    if errors.Is(err, gorm.ErrRecordNotFound) {
        return r.db.Create(a).Error
    }
    a.ID = existing.ID
    return r.db.Model(&existing).Updates(a).Error
    ```

    **VisualExamRepository**, **ClinicalDiagnosisRepository**, **ClinicalPrescriptionRepository**, **FollowUpRepository**:
    Same pattern as `AnamnesisRepository` — `GetByClinicalRecordID` + `Upsert`.

    Every `GetBy*` must convert `gorm.ErrRecordNotFound` → `*domain.ErrNotFound`.
    Every struct must have a `New*Repository(db *gorm.DB) *Repository` constructor.

    ```go
        records       domain.ClinicalRecordRepository

        visualExams   domain.VisualExamRepository
        diagnoses     domain.ClinicalDiagnosisRepository
        prescriptions domain.ClinicalPrescriptionRepository
        appointments  domain.AppointmentRepository

    ```

    **DTOs** (one per upsert operation):
    - `CreateRecordInput { AppointmentID uint, RecordType domain.RecordType }`
    - `AnamnesisInput` — mirrors `domain.Anamnesis` fields (no ID, no ClinicalRecordID — those are set by service)
    - `VisualExamInput` — mirrors `domain.VisualExam` fields
    - `DiagnosisInput` — mirrors `domain.ClinicalDiagnosis` fields
    - `PrescriptionInput` — mirrors `domain.ClinicalPrescription` fields

    **Methods**:

    ```go
    // CreateRecord creates a new ClinicalRecord for an appointment.
    // Returns error if appointment already has a record (unique constraint).
    func (s *Service) CreateRecord(clinicID uint, input CreateRecordInput) (*domain.ClinicalRecord, error)

    // UpsertAnamnesis creates or updates the anamnesis step.
    // Validates clinical record belongs to the appointment.
    func (s *Service) UpsertAnamnesis(recordID uint, clinicID uint, input AnamnesisInput) (*domain.Anamnesis, error)

    // UpsertVisualExam creates or updates the visual exam step.
    func (s *Service) UpsertVisualExam(recordID uint, clinicID uint, input VisualExamInput) (*domain.VisualExam, error)

    // UpsertDiagnosis creates or updates the diagnosis step.
    func (s *Service) UpsertDiagnosis(recordID uint, clinicID uint, input DiagnosisInput) (*domain.ClinicalDiagnosis, error)

    // UpsertPrescription creates or updates the prescription step.
    // Auto-computes ValidUntil = now + ValidityMonths.
    func (s *Service) UpsertPrescription(recordID uint, clinicID uint, input PrescriptionInput) (*domain.ClinicalPrescription, error)

    // SignAndComplete marks the record as signed and the appointment as completed.
    // Validates: Anamnesis + VisualExam + Diagnosis + Prescription all exist.
    // Sets: SignedAt, SignedByID, CUPS = 890205, status = signed.
    // Then calls appointmentRepo.Update to set completed status.
    func (s *Service) SignAndComplete(recordID uint, specialistID uint, professionalTP string) (*domain.ClinicalRecord, error)

    // GetByAppointmentID returns the full clinical record with all sub-entities.
    func (s *Service) GetByAppointmentID(appointmentID uint) (*domain.ClinicalRecord, error)
    ```

    `SignAndComplete` business rules:
    - Fetch the record; return 404 if not found
    - Fetch each sub-entity; if any is nil return `ErrValidation{Message: "step X must be completed before signing"}`
    - Set `SignedAt = &now`, `SignedByID = &specialistID`, `CUPS = "890205"`, `Status = RecordStatusSigned`
    - Call `s.records.UpdateStatus(...)`
    - Log: `s.logger.Info("clinical record signed", zap.Uint("record_id", record.ID), zap.Uint("specialist_id", specialistID))`

    `CreateRecord` must set CUPS automatically: `890205` for `new_consultation`, `890307` for `follow_up`.
