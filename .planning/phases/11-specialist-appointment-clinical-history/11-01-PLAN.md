# Plan 11-01: Go Backend — Domain Models + DB Migrations

**Phase:** 11 — Specialist Appointment & Clinical History Flow
**Plan:** 01 of 08
**Wave:** 1
**Depends on:** Nothing (first plan of phase)
**Estimated scope:** 9 new files + 2 modified files

---

## Goal

Define the 6 new domain entities for the structured clinical record system
(`ClinicalRecord`, `Anamnesis`, `VisualExam`, `ClinicalDiagnosis`,
`ClinicalPrescription`, `FollowUp`), add `StartedAt` / `CompletedAt` to
`Appointment`, write all SQL migration files, and register the entities in
`AutoMigrate` (local-only).

This plan produces zero runnable code yet — it is the schema and contract
layer that every subsequent plan depends on.

---

## Success Criteria

- [ ] `internal/domain/clinical_record.go` exists with all 6 entity structs + 6 repository interfaces
- [ ] `internal/domain/appointment.go` has `StartedAt *time.Time` and `CompletedAt *time.Time` fields
- [ ] 8 SQL migration files exist under `db/migrations/platform/` (000005–000012)
- [ ] All new tables include `clinic_id` as second column, `deleted_at TIMESTAMPTZ NULL`, `set_updated_at` trigger, and a partial index on `(clinic_id) WHERE deleted_at IS NULL`
- [ ] `internal/platform/storage/postgres/db.go` AutoMigrate list includes all 6 new structs
- [ ] `make build` passes with 0 errors

        Appointment  *Appointment          `json:"appointment,omitempty"  gorm:"foreignKey:AppointmentID"`

    5. **VisualExam struct** (step 2, shared by both archetypes — full field list from CONTEXT.md §6.1):
    Include all AV (sc/cc × dist/near × OD/OI), Autoref, Subj refraction, Keratometry, IOP, Biomicroscopy anterior (cornea/lens/conj/AC per eye), Fundus posterior (vitreous/disc/macula/periph per eye), OcularMotility, Notes fields.
    Use `decimal(5,2)` for float optic values and `varchar(10)` for Snellen strings.

    6. **ClinicalDiagnosis struct** (step 3, new consultation):
    Include PrimaryCode (varchar 10), PrimaryDescription (varchar 200), DiagnosisType (int 1/2/3), Related1/2/3 Code+Desc, CIE11Code (varchar 20 nullable), OpticalCorrectionPlan (varchar 50), PatientEducation (text), NextControlDate (*time.Time), NextControlReason (varchar 50), RequiresReferral (bool), CUPS (varchar 10).

    7. **ClinicalPrescription struct** (step 4, legal document):
    Include OD+OI fields (SphOD/I, CylOD/I, AxisOD/I, AVccOD/I, AddOD/I, DPOD/I all decimal(5,2) or int for axis/varchar for AV), LensType/Material/Use (varchar 50), MountingHeight (decimal 5,2), Treatments (text — JSON array), ValidityMonths (int default 12), ValidUntil (*time.Time), SignedByID (*uint), SignedAt (*time.Time), ProfessionalTP (varchar 50), CUPS (varchar 10).

    8. **FollowUp struct** (all 4 follow-up steps combined):
    Include Step1 fields (ControlReason text, CorrectionSatisfaction varchar 20, SubjectiveChanges/NewMedications/SystemicChanges/CorrectionUsage/DailyUsageHours/ObsBeforeExam), Step3 fields (DiagnosticEvolution varchar 20, EvolutionDescription text, NewDiagCode varchar 10, NewDiagDesc varchar 200, OpticalDecision varchar 30, NextControlDate *time.Time, NextControlInterval varchar 50, PatientEducation text), Step4 fields (FormulaDecision varchar 20, NewValidityMonths int default 12, NewValidUntil *time.Time, SignedByID *uint, SignedAt *time.Time, ProfessionalTP varchar 50, CUPS varchar 10).

  <action>
    Create 6 migration pairs following the pattern below. Every table must have:
    - `id BIGSERIAL PRIMARY KEY`
    - `clinic_id INTEGER NOT NULL REFERENCES clinics(id)` as second column
    - `deleted_at TIMESTAMPTZ NULL`
    - `created_at / updated_at TIMESTAMPTZ NOT NULL DEFAULT now()`
    - `CREATE TRIGGER set_updated_at BEFORE UPDATE ON <table> FOR EACH ROW EXECUTE FUNCTION trigger_set_updated_at();`
    - `CREATE INDEX CONCURRENTLY IF NOT EXISTS idx_<table>_clinic_active ON <table>(clinic_id) WHERE deleted_at IS NULL;`
    - All CHECK constraints on enum varchar columns

    **000006** — `clinical_records`:
    ```sql
    CREATE TABLE IF NOT EXISTS clinical_records (
        id              BIGSERIAL PRIMARY KEY,
        clinic_id       INTEGER NOT NULL REFERENCES clinics(id),
        appointment_id  BIGINT NOT NULL UNIQUE REFERENCES appointments(id),
        patient_id      BIGINT NOT NULL REFERENCES patients(id),
        specialist_id   BIGINT NOT NULL REFERENCES users(id),
        record_type     VARCHAR(20) NOT NULL CHECK (record_type IN ('new_consultation','follow_up')),
        status          VARCHAR(20) NOT NULL DEFAULT 'in_progress' CHECK (status IN ('in_progress','completed','signed')),
        cups            VARCHAR(10),
        signed_at       TIMESTAMPTZ,
        signed_by_id    BIGINT REFERENCES users(id),
        created_at      TIMESTAMPTZ NOT NULL DEFAULT now(),
        updated_at      TIMESTAMPTZ NOT NULL DEFAULT now(),
        deleted_at      TIMESTAMPTZ
    );
    CREATE TRIGGER set_updated_at BEFORE UPDATE ON clinical_records FOR EACH ROW EXECUTE FUNCTION trigger_set_updated_at();
    CREATE INDEX CONCURRENTLY IF NOT EXISTS idx_clinical_records_clinic_active ON clinical_records(clinic_id) WHERE deleted_at IS NULL;
    CREATE INDEX IF NOT EXISTS idx_clinical_records_appointment ON clinical_records(appointment_id);
    CREATE INDEX IF NOT EXISTS idx_clinical_records_patient ON clinical_records(patient_id);
    ```

    **000007** — `anamneses`:
    ```sql
    CREATE TABLE IF NOT EXISTS anamneses (
        id                          BIGSERIAL PRIMARY KEY,
        clinic_id                   INTEGER NOT NULL REFERENCES clinics(id),
        clinical_record_id          BIGINT NOT NULL UNIQUE REFERENCES clinical_records(id),
        reason_for_visit            TEXT NOT NULL,
        systemic_background         TEXT,
        ocular_background           TEXT,
        family_background           TEXT,
        pharmacological_background  TEXT,
        current_correction_type     VARCHAR(50),
        current_correction_time     VARCHAR(100),
        created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
        updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
    );
    CREATE TRIGGER set_updated_at BEFORE UPDATE ON anamneses FOR EACH ROW EXECUTE FUNCTION trigger_set_updated_at();
    CREATE INDEX IF NOT EXISTS idx_anamneses_clinic ON anamneses(clinic_id);
    ```

    All AV, Autoref, Subj, Kerato, IOP, Biomicroscopy, Fundus, OcularMotility, Notes columns.
