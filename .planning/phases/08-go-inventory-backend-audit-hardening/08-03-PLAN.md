  <title>Rewrite AdjustStock as AdjustStockByItemID — operates on specific item ID with SELECT FOR UPDATE in a transaction</title>
  <read_first>

    Remove the existing `AdjustStock` method entirely (L581–610). Add the following replacement method:

    ```go
    func (s *Service) AdjustStockByItemID(itemID uint, delta int, reason string) (*domain.InventoryItem, error) {
        var result *domain.InventoryItem
        err := s.db.Transaction(func(tx *gorm.DB) error {
            var item domain.InventoryItem
            if err := tx.Clauses(clause.Locking{Strength: "UPDATE"}).
                First(&item, itemID).Error; err != nil {
                if errors.Is(err, gorm.ErrRecordNotFound) {
                    return &domain.ErrNotFound{Resource: "inventory_item"}
                }
                return err
            }
            newQty := item.Quantity + delta
            if newQty < 0 {
                return &domain.ErrValidation{
                    Field:   "delta",
                    Message: "el ajuste resultaría en stock negativo",
                }
            }
            if err := tx.Model(&item).Update("quantity", newQty).Error; err != nil {
                return err
            }
            item.Quantity = newQty
            result = &item
            return nil
        })
        if err != nil {
            return nil, err
        }
        return s.itemRepo.GetByID(result.ID)
    }
    ```

    Ensure `"errors"` is imported (already present after 08-02-T2). The `gorm` and `clause` imports were added in 08-02-T1.

    The `reason` parameter is accepted but not persisted to a separate audit table (no such table exists in the current schema). It is logged via the service logger:
    ```go
    s.logger.Info("stock adjusted", zap.Uint("item_id", itemID), zap.Int("delta", delta), zap.String("reason", reason))
    ```
    Add `"go.uber.org/zap"` if not already imported (it is — it was already in service.go).
