# Plan 10-05: Receptionist entity lists — LabOrders, Patients, Quotes, Sales, DiscountRequests, CashRegisterHistory, DailyReportHistory

**Phase:** 10 — Unified Table Design — Entity Tables
**Plan:** 05 of 05
**Wave:** 2
**Depends on:** Plan 10-01 (EntityTable defaults must be updated first)
**Estimated scope:** 7 pages to migrate

---

## Goal

Migrate all receptionist entity list pages to use the centralized `EntityTable` component. Receptionist pages are mirrors of admin pages but with different role-based routing and limited action sets. The migration follows the same pattern: remove custom pagination state, replace DataTable + custom pagination with EntityTable.

- [ ] `receptionist/LabOrders.tsx` uses `<EntityTable>`
