- [ ] Migraciones ejecutan sin error con `php artisan migrate`
- [ ] `CashRegisterClose` + `CashRegisterClosePayment` + `CashCountDenomination` models creados con relaciones correctas

- [ ] `CashRegisterCloseController` responde a todos los endpoints con los códigos HTTP correctos

- [ ] `POST /api/v1/cash-register-closes/{id}/approve` solo accesible para admin

    - counted_amount (decimal:2 default 0)
    - difference (decimal:2) — computed: registered - counted
    - timestamps

    **cash_count_denominations:**
    - id

    - denomination (integer) — valor en pesos: 100000, 50000, 20000, 10000, 5000, 2000, 1000, 500, 200, 100, 50
    - quantity (integer default 0)
    - subtotal (decimal:2) — computed: denomination * quantity
    - timestamps

<task type="sequential">
  <name>Create CashRegisterCloseService</name>
  <files>app/Services/CashRegisterCloseService.php</files>
  <action>
    Crear service con estos métodos:

    **createWithDetails(array $validated, int $userId): CashRegisterClose**
    - Crea el CashRegisterClose con user_id = $userId, status = 'draft'
    - Itera sobre los payment_methods del request y crea CashRegisterClosePayment por cada uno
    - Calcula difference = registered_amount - counted_amount por cada método
    - Crea CashCountDenomination por cada denominación (si se envían)
    - Llama recalculateTotals() al final

    **recalculateTotals(CashRegisterClose $close): void**
    - total_registered = sum(payments.registered_amount)
    - total_counted = sum(payments.counted_amount)
    - total_difference = total_registered - total_counted
    - Actualiza el registro

    **updateWithDetails(CashRegisterClose $close, array $validated): CashRegisterClose**
    - Solo permite si status === 'draft'
    - Sincroniza payments (delete all → recreate) con recálculo de diferencias
    - Sincroniza denominations (delete all → recreate con subtotals)
    - Llama recalculateTotals()

    **submit(CashRegisterClose $close): CashRegisterClose**
    - Valida que status === 'draft'
    - Cambia status a 'submitted'
    - Guarda y retorna

    **approve(CashRegisterClose $close, int $adminId, ?string $notes): CashRegisterClose**
    - Valida que status === 'submitted'
    - Cambia status a 'approved', approved_by = $adminId, approved_at = now(), admin_notes = $notes
    - Guarda y retorna
  </action>

    **CashRegisterCloseController:**

    Constructor inyecta CashRegisterCloseService.

    ```php
    // index — admin ve todos, receptionist solo los suyos
    public function index(Request $request)
    {
        $query = CashRegisterClose::with(['user', 'approvedBy'])
            ->apiFilter($request);

        if (!auth()->user()->hasRole('admin')) {
            $query->where('user_id', auth()->id());
        }

        $perPage = min(max(1, (int)$request->get('per_page', 15)), 100);
        return new CashRegisterCloseCollection($query->paginate($perPage));
    }

    // show — con payments y denominations eager loaded
    public function show($id)
    {
        $close = CashRegisterClose::with(['user', 'approvedBy', 'payments', 'denominations'])
            ->findOrFail($id);
        $this->authorize('view', $close); // Policy: admin o dueño
        return new CashRegisterCloseResource($close);
    }

    // store
    public function store(StoreCashRegisterCloseRequest $request)
    {
        $close = $this->service->createWithDetails($request->validated(), auth()->id());
        return new CashRegisterCloseResource($close->load(['payments', 'denominations']));
    }

    // update — solo draft
    public function update(UpdateCashRegisterCloseRequest $request, $id)
    {
        $close = CashRegisterClose::findOrFail($id);
        $this->authorize('update', $close);
        $close = $this->service->updateWithDetails($close, $request->validated());
        return new CashRegisterCloseResource($close->load(['payments', 'denominations']));
    }

    // submit — POST /cash-register-closes/{id}/submit
    public function submit($id)
    {
        $close = CashRegisterClose::findOrFail($id);
        $this->authorize('update', $close);
        return new CashRegisterCloseResource($this->service->submit($close));
    }

    // approve — POST /cash-register-closes/{id}/approve (admin only)
    public function approve(ApproveCashRegisterCloseRequest $request, $id)
    {
        $close = CashRegisterClose::findOrFail($id);
        return new CashRegisterCloseResource(
            $this->service->approve($close, auth()->id(), $request->admin_notes)
        );
    }
    ```
