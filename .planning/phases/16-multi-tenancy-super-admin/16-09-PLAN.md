# 16-09 — Fix Local Dev Tenant Routing (Gaps 1, 2, 3)

## Context

The `TenantFromSubdomain` middleware has a local dev fast path (APP_ENV=local) that:
1. Reads `DEFAULT_TENANT_SLUG` from env (defaults to "main")
2. Hard-codes `schemaNameKey = "optica_" + slug` for ALL slugs
3. Hard-codes `opticaIDKey = uint(0)` always

This breaks two things in local development:
- **Gap 1 (Blocker):** The `admin` slug is never routed to the `platform` schema, so
  `superadmin@convision.com` login always goes to `loginTenantUser` (optica_main) and gets 401.
- **Gap 2 (Major):** `optica_id=0` means the auth service can't look up feature flags from
  `FeatureCache`, so `feature_flags` is always `[]` in the JWT.
- **Gap 3 (Major):** Because feature_flags is empty, admin nav feature gating is untestable.

The fix mirrors what the production path already does correctly (lines 43-62).

## Must Haves (goal-backward verification)

- `DEFAULT_TENANT_SLUG=admin` env var causes the middleware to set `schema_name=platform`
- `DEFAULT_TENANT_SLUG=main` env var causes the middleware to set `schema_name=optica_main` AND look up optica_id from cache
- `superadmin@convision.com` login succeeds locally when `DEFAULT_TENANT_SLUG=admin` and returns a JWT with `role=super_admin`
- A regular tenant login (admin@convision.com) with `DEFAULT_TENANT_SLUG=main` returns `feature_flags` with actual values (not empty `[]`)
- `.env.example` documents `DEFAULT_TENANT_SLUG` with usage instructions
- `make build` succeeds

```go
func TenantFromSubdomain(cache *opticacache.Cache, baseDomain string) gin.HandlerFunc {
    return func(c *gin.Context) {
        if os.Getenv("APP_ENV") == "local" {
            slug := os.Getenv("DEFAULT_TENANT_SLUG")
            if slug == "" {
                slug = "main"
            }
            if slug == "admin" {
                c.Set(schemaNameKey, "platform")
                c.Set(opticaIDKey, uint(0))
                c.Next()
                return
            }
            schemaName := "optica_" + slug
            opticaID := uint(0)
            if entry, ok := cache.GetBySlug(slug); ok {
                opticaID = entry.ID
            }
            c.Set(schemaNameKey, schemaName)
            c.Set(opticaIDKey, opticaID)
            c.Next()
            return
        }

        host := c.GetHeader("X-Forwarded-Host")
        if host == "" {
            host = c.Request.Host
        }
        host = strings.Split(host, ":")[0]

        slug := extractSlug(host, baseDomain)
        if slug == "" {
            c.AbortWithStatusJSON(http.StatusBadRequest, gin.H{"message": "subdominio inválido"})
            return
        }

        if slug == "admin" {
            c.Set(schemaNameKey, "platform")
            c.Set(opticaIDKey, uint(0))
            c.Next()
            return
        }

        entry, ok := cache.GetBySlug(slug)
        if !ok {
            c.AbortWithStatusJSON(http.StatusNotFound, gin.H{"message": "óptica no encontrada"})
            return
        }
        if !entry.IsActive {
            c.AbortWithStatusJSON(http.StatusForbidden, gin.H{"message": "óptica inactiva"})
            return
        }

        c.Set(schemaNameKey, entry.SchemaName)
        c.Set(opticaIDKey, entry.ID)
        c.Next()
    }
}
```
